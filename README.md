# DSA Sorting and Searching Algorithm Analysis System

## Module
ITS60504 - Data Structures and Algorithms

## Project Description
This project is a C++ console application that implements and analyzes sorting and searching algorithms.

## Algorithms Implemented

### Sorting Algorithms
- Selection Sort
- Quick Sort
- Merge Sort

### Searching Algorithms
- Binary Search
- Interpolation Search

## Program Features
- Generates 100+ random records
- Displays all records
- Sorts records in ascending order
- Searches for user-entered number
- Measures execution time
- Supports multiple input sizes
